About Sentence Checker Tool

The sentence checker tool is a simple script that accepts a sentence as input and identified its type. The result is display in the output box.

How to use Sentence Checker Tool

It is very easy to use this tool. Either you enter a single sentence by typing it or paste it from some other source. Then click the button "Check Sentence Type". The result is computed and displayed immediatly. If you want to delete or clear the textarea, simply click the "Refresh" button. Note that the information will permanantly disappear if click refresh; therefore, I recommend that you keep a copy of all the data before using it here.

Types of Sentences

There are three types of sentence in English language than we try to identify using this script. They are listed below.

  • SIMPLE SENTENCE
  • COMPOUND SENTENCE
  • COMPLEX SENTENCE
